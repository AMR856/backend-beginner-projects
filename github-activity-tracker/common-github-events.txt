PushEvent	Someone pushed commits to a branch.
PullRequestEvent	A pull request was opened, closed, merged, etc.
IssuesEvent	An issue was opened, closed, edited, etc.
IssueCommentEvent	A comment was added to an issue or PR.
WatchEvent	Someone starred a repository.
ForkEvent	Someone forked a repository.
CreateEvent	A new branch or repository was created.
DeleteEvent	A branch or tag was deleted.
ReleaseEvent	A new release was published.
PullRequestReviewEvent	A PR review was submitted.
PublicEvent	A repository was made public.
MemberEvent	A user was added or removed from a repository.
